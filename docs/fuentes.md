# Fuentes de Datos

## Datos Abiertos Colombia — Delitos

- **URL:** https://datos.gov.co
- **Entidad:** Policía Nacional / Ministerio de Defensa
- **Formato:** CSV / API SODA
- **Frecuencia de actualización:** Mensual (aproximado)
- **Cobertura temporal:** 2010 - presente
- **Notas:** Los dataset IDs pueden cambiar; verificar en el portal.

## DANE — DIVIPOLA

- **URL:** https://www.dane.gov.co/index.php/servicios-al-ciudadano/servicios-informacion/divipola
- **Descripción:** División Político-Administrativa de Colombia
- **Formato:** Excel / CSV
- **Uso:** Estandarización de códigos y nombres de municipios

## DANE — Proyecciones de Población

- **URL:** https://www.dane.gov.co/index.php/estadisticas-por-tema/demografia-y-poblacion/proyecciones-de-poblacion
- **Descripción:** Proyecciones de población municipal basadas en el Censo 2018
- **Formato:** Excel
- **Uso:** Denominador para cálculo de tasas per cápita

## GeoJSON — Geometrías Municipales

- **Fuente:** Repositorios abiertos de geometrías de Colombia
- **Formato:** GeoJSON
- **Uso:** Polígonos para mapas coropléticos
- **Nota:** Las geometrías se simplifican para optimizar performance en el dashboard
